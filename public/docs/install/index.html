Installation

This page will help you install gb.

Prerequisites

gb is compatible with Go 1.4. If you haven’t installed Go yet, we recommmend using the binary installers available from the office Go website.

Installation

gb can be installed with the following command

go get github.com/constabulary/gb/...

Keeping up to date

gb is under active development, you can upgrade your gb installation to the latest with the following command

go get -u github.com/constabulary/gb/...
